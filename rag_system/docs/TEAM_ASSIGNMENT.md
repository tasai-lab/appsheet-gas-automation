## 🎯 役割別スキルセット要件

### Backend開発者

**必須スキル:**
- Python 3.11+ （FastAPI経験）
- SQL（MySQL 9.0、Vector Type知識があれば尚可）
- Vertex AI / Gemini API経験
- 非同期プログラミング（asyncio）

**推奨スキル:**
- データベース設計経験
- データ移行経験
- RAG/LLMアプリケーション開発経験

---

### Frontend開発者

**必須スキル:**
- TypeScript
- Next.js 14（App Router）
- React Hooks
- Tailwind CSS + shadcn/ui

**推奨スキル:**
- SSE（Server-Sent Events）経験
- リアルタイムUI実装経験
- アクセシビリティ対応経験

---

### DevOps

**必須スキル:**
- GCP（Cloud SQL, Cloud Run）
- MySQL運用経験
- CI/CD（GitHub Actions）

**推奨スキル:**
- Cloud SQL for MySQL 9.0経験
- Vector Database運用経験
- パフォーマンステスト経験

---

### QA

**必須スキル:**
- E2Eテスト経験（Cypress等）
- API テスト経験

**推奨スキル:**
- パフォーマンステスト経験
- セキュリティテスト経験

## 📞 エスカレーションパス

### 技術的問題

**Backend関連** → Backend Lead → PM → CTO
**Frontend関連** → Frontend Lead → PM → CTO
**インフラ関連** → DevOps Lead → PM → CTO

**遅延発生** → 担当者 → PM → ステークホルダー
**リスク顕在化** → 担当者 → PM → リスク管理委員会

### 予算問題

**コスト超過懸念** → PM → 財務担当 → 承認者
